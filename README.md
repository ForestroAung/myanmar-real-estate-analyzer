#  Myanmar Real Estate Price Analysis

##  Overview

This project analyzes real estate data in Myanmar to understand property price trends and key factors affecting housing prices.

The goal is to transform raw property data into meaningful insights using data analysis and machine learning techniques.

---

##  Objectives

* Analyze relationships between property features and price
* Compare property types (e.g., apartments vs landed houses)
* Identify key factors influencing real estate prices
* Visualize trends using charts and graphs

---

##  Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn

---

##  Dataset

The dataset contains information such as:

* Number of rooms
* Number of bedrooms
* Property type
* Price

(Note: Dataset is not uploaded due to size or privacy constraints.)

---

##  Key Analysis

### 1. Data Cleaning

* Removed duplicates
* Handled missing values
* Converted categorical variables

### 2. Exploratory Data Analysis (EDA)

* Distribution of prices
* Price comparison by property type
* Correlation between features

### 3. Visualization

* Bar charts for property types
* Scatter plots for rooms vs price
* Heatmap for feature relationships

---

##  Key Insights

* Property type significantly impacts price
* More rooms do not always mean higher prices
* Landed houses tend to be more expensive than flats

---

##  Machine Learning 

* Built regression models to predict housing prices
* Observed overfitting in some models, indicating need for better generalization
